# 위험 단계 정의
| 단계       | 의미        |
| -------- | --------- |
| SAFE     | 일반 감정 상태  |
| LOW      | 약한 위험 신호  |
| MEDIUM   | 반복적 위험 표현 |
| HIGH     | 자살/자해 암시  |
| CRITICAL | 즉시 개입 필요  |

# 표현 예시
| 단계       | 예시 표현        |
| -------- | ------------ |
| LOW      | “지친다”        |
| MEDIUM   | “사라지고 싶다”    |
| HIGH     | “죽고 싶다”      |
| CRITICAL | “오늘 끝내려고 한다” |

# 단계별 AI 행동
| 단계       | AI 행동             |
| -------- | ----------------- |
| SAFE     | 일반 상담 유지          |
| LOW      | 감정 확인             |
| MEDIUM   | 추가 확인 질문          |
| HIGH     | 안전 확인 + 위기 리소스 안내 |
| CRITICAL | 즉시 위기 프로토콜        |

# 우선순위 규칙
- 위험 표현 감지 시 설문 점수보다 실시간 대화 내용을 우선 적용한다.
- 단일 키워드만으로 CRITICAL 판정을 내리지 않는다.
- 문맥 분석 결과를 함께 고려한다.
