# GAD-7 기준
| 점수    | 상태     |
| ----- | ------ |
| 0~4   | 최소 불안  |
| 5~9   | 경미 불안  |
| 10~14 | 중등도 불안 |
| 15~21 | 심한 불안  |

# 상태 분류 규칙
| PHQ    | GAD   | 상태 분류    |
| ------ | ----- | -------- |
| 낮음     | 낮음    | 일반 감정 케어 |
| 높음     | 낮음    | 우울 중심    |
| 낮음     | 높음    | 불안 중심    |
| 둘 다 높음 | 복합 위험 |          |
| 매우 높음  | 매우 높음 | 고위험군     |

# 상태별 AI 대응 전략
| 상태       | AI 전략            |
| -------- | ---------------- |
| 일반 감정 케어 | 공감 + 가벼운 정서 지원   |
| 우울 중심    | 감정 반영 + 부담 최소화   |
| 불안 중심    | 안정화 + 호흡/정리      |
| 복합 위험    | 자극 최소화 + 짧은 질문   |
| 고위험군     | 안전 확인 + 위기 대응 우선 |

# 우선순위 규칙
- PHQ와 GAD가 모두 높은 경우 복합 위험 상태를 우선 적용한다.
- 위험 표현 감지 시 설문 점수보다 위기 프로토콜을 우선 적용한다.
- 최근 대화 내 자살/자해 표현이 있을 경우 고위험군으로 재분류 가능하다.
